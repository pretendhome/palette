# Nora Learning Bot

Voice-first adaptive learning system for children with learning differences. Uses the ARON anonymized avatar pattern from the La Scuola pilot.

## Architecture

- **Session structure**: A (warm-up) -> B (core learning) -> A (cool-down)
- **Tiered tools**: Universal -> Targeted -> Intensive
- **The Italian Bridge**: Bilingual identity as cognitive scaffold, not scheduling constraint

## Key Insight

Oral performance 2+ levels above written for 2 consecutive semesters = flag for phonological screening. Costs nothing. Requires no technology. Catches learning differences 2+ years earlier.
