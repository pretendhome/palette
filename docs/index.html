About

Most AI products send your work to someone else's server. Mission Canvas doesn't. It runs on your machine, routes every query through a classification layer, and stores your decisions as a judgment trail that compounds over time.

Legal is the proof. In February 2026, a federal court showed how uncontrolled AI use can put privilege at risk. Mission Canvas is the architectural answer: local by default, external only after the boundary is clear.

The same pattern applies anywhere the useful context is also the dangerous context: medical, finance, accounting, startup strategy.

Built for work that cannot leak

Your client data never leaves this machine. Not sent to a server. Not stored in a cloud. Not shared with any AI provider. Classification happens locally.

External search is governed, not automatic. When you need public information, Mission Canvas keeps client-specific strategy local, opens a governed research route, and integrates results back under your control.

Every decision is logged, not forgotten. An immutable audit trail records what was asked, what was retrieved, what model answered, and what boundary applied.

Technical: socket firewall · 3-layer PII sanitizer · governance tiers · append-only decision log · approval workflows · zero CVEs

Get started in 60 seconds

Step 1: Install Ollama (required — runs AI locally on your machine)

→ Download Ollama  then run:  ollama pull qwen2.5:7b

Step 2: Install Mission Canvas

docker build -t mission-canvas https://github.com/pretendhome/mission-canvas.git
docker run -d -p 7891:7891 mission-canvas
curl -fsSL https://missioncanvas.ai/install.sh | sh
irm https://missioncanvas.ai/install.ps1 | iex

Open http://localhost:7891 → connect Ollama → start asking questions.

  • 174 classified problem types
  • 148 knowledge entries with evidence tiers
  • PII gateway — nothing sensitive leaves without clearance
  • Append-only decision log
  • Multi-model routing: Ollama (local) + Claude/GPT (optional, for non-sensitive queries)

No Python, Git, or Homebrew needed. Just Ollama + one install command.
