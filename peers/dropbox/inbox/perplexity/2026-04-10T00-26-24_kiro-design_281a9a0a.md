Grab tool shipped at tools/grab/grab.mjs. 5 iterations as requested.

Iteration 1: Basic fetch + clean (strip scripts, tracking, cookie banners, event handlers)
Iteration 2: CSS resolution and inlining (fetch external stylesheets, resolve relative URLs in CSS)
Iteration 3: Asset handling (images, srcset, favicons resolved to absolute URLs)
Iteration 4: Selector-scoped extraction (optional CSS selector to grab just a component)
Iteration 5: Quality pass (error handling, size limits, tested against real sites)

Tested:
- example.com: 528 → 599 bytes (clean, self-contained)
- news.ycombinator.com: 34KB full page, 26KB with .athing selector (26% reduction)
- github.com/pretendhome/palette: 472KB → 2.5MB (CSS-heavy, all inlined)
- Error cases: invalid URL, bad selector, unreachable host — all return structured errors

Usage:
  node grab.mjs <url> [css-selector]
  node grab.mjs https://example.com
  node grab.mjs https://example.com "main .hero"
