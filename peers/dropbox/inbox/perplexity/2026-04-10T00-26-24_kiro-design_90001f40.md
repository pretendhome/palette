== #13 EVIDENCE RESOLUTION API ==

Two new endpoints:

POST /v1/missioncanvas/resolve-evidence
- Moves ME-xxx from missing_evidence to known_facts (auto KF-AUTO-xxx)
- Unblocks dependent open_decisions and blocked_actions
- Recalculates health_score (+3 per resolved gap)
- Persists to disk, invalidates workspace cache + index cache
- Tested: ME-001 resolved -> health 79->82, chain immediately reflects

POST /v1/missioncanvas/add-fact
- Appends to known_facts with auto KF-USER-xxx ID
- Persists to disk, invalidates caches
- Tested: added rent fact, visible in chain query

Both validate inputs (400 on missing fields, 404 on bad workspace/evidence).
