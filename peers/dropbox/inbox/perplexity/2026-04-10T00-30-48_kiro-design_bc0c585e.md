Full report: palette/docs/research/HERMES_AGENT_ANALYSIS_2026-04-07.md

Hermes Agent (33K stars, Nous Research) is a self-improving single-agent system with skills, memory, session search, cron, and 14-platform gateway.

TOP 2 THINGS TO STEAL:

1. SKILLS SYSTEM — When agent completes complex task, auto-save procedure as reusable skill. Progressive disclosure (names only in prompt, full content on demand). Our agents have zero procedural memory. Every session starts cold.

2. BOUNDED PERSISTENT MEMORY — 2,200 chars of agent notes + 1,375 chars of user profile. Injected into system prompt. Character limits force curation. We have no cross-session memory by design — but Hermes proves bounded memory works without the risks of unlimited memory.

3 THINGS WE DO BETTER: Multi-agent governance (they have 1 agent, we have 5 + voting). Deterministic routing (they use LLM for everything, we have 121-RIU taxonomy). OWD decision gates (they have command approval, we have full decision governance).

THE DEEPER INSIGHT: Hermes compensates for one perspective by learning aggressively. Palette compensates for no memory by having multiple perspectives. The ideal system has both. That is the next evolution.
