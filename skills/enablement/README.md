# Enablement Skill

Agentic enablement coaching system. Guides learners through building their own AI toolkit — customized AI assistants that remember context and improve over time.

## Files

- `enablement-coach.md` — Full coaching system (intake protocol, 7-stage path, session loop)
- `agentic-enablement-skill.md` — Skill definition and constraints

## Usage

The enablement coach runs as an interactive session. It profiles the learner, proposes next steps, and advances them through stages based on demonstrated capability. See `enablement/CLAUDE.md` for Claude Code integration.
